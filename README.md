# Описание

На страничке одна только кнопка, по нажатию на которую появляется вспывающее окно (модальное), а в модальном окне есть кнопка - закрыть, которая закрывает это окно. 

### Start development
Run in development mode **http://localhost:9000**

### Build
Build project

```
$ npm run build

```

```
$ npm start

```
